# 📚 Student-Course Management System

A Spring Boot web application to manage Students and Courses with Create, Read, and Update operations.

---

## 🚀 Features

- ➕ Add new students  
- 📖 View all students with course details  
- ✏️ Update existing student information  
- 🔗 Inner Join between Student and Course  
- 🎨 Styled UI using Bootstrap  

---

## 🛠️ Technologies Used

- Java  
- Spring Boot  
- Spring MVC  
- Spring Data JPA  
- MySQL  
- JSP & JSTL  
- Maven  
- Bootstrap  

---

## 🧩 Project Structure

- **Controller** → Handles HTTP requests  
- **Service** → Business logic  
- **Repository** → Database interaction  
- **Entity** → JPA models  
- **JSP** → Frontend views  

---

## ⚙️ Setup Instructions

### 1. Clone the repository

```bash
git clone https://github.com/abhaysingh-oo7/Spring_Project_Sga.git
```

### 2. Configure database in `application.properties`

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/spring_project
spring.datasource.username=root
spring.datasource.password=YOUR_PASSWORD
```

### 3. Run the application

```bash
mvn spring-boot:run
```

### 4. Open in browser

```
http://localhost:8080/students
```
